# Llinas-Lab-Codes
This is a repository for the codes used in data analysis from the Llinás lab at Penn State


# Hemoglobin Peptide Calling RScript
The RScript is written to call putative hemoglobin dervied peptides from MS data collected in positive and negative mode. In order for the script to work, the AAreferenceChart.xlsx and the Peptides_01.txt files need to be saved into the same directory.
